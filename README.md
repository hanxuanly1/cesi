# 产品加工统计管理系统

## 功能特点

- **入库管理**：编号入库，客户与编号强制绑定
- **出库管理**：按编号出库，自动验证客户归属
- **库存查询**：实时查看库存情况
- **残次品登记**：记录残次品数量和原因
- **财务报表**：月度数据汇总，支持导出

## 安装步骤

1. 安装依赖：
```bash
npm run install-all
```

2. 启动服务：
```bash
npm run dev
```

3. 访问系统：
- 前端：http://localhost:5173
- 后端：http://localhost:3000

## 使用说明

1. 首先在"基础设置"中添加客户和产品
2. 在"入库管理"中登记来料，系统会自动生成编号绑定
3. 在"出库管理"中按编号出库，系统会验证客户归属
4. 在"库存查询"中查看实时库存
5. 在"残次品登记"中记录残次品
6. 在"财务报表"中查看月度汇总并导出

## 技术栈

- 前端：Vue 3 + Element Plus + Vite
- 后端：Node.js + Express
- 数据库：SQLite
